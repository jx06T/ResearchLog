## 研究筆記
### 透過以下工具搭建簡易日誌管理系統
- obsidian - 文件管理
- obsidian plugin
	- Kanban - 任務看板
	- QuickAdd - 快速紀錄
	- Templater - 進階模板
	- Keyshots - 保留 vs code 習慣之快捷鍵
### 並實現以下功能
1. 透過 Kanban 將任務分類為以下類別
	- 可能的探索方向
	- 待執行任務
	- 執行中任務
	- 取得結果之任務
	- 無結果之任務
2. 可以透過卡片直接套用模板創建該任務記錄
3. 透過 QuickAdd 以及 Templater 實現以下三種簡易記錄方式
	- 日誌：檔名為當天日期並套用日誌模板
	- 週記：檔名為使用者自訂，並透過 Templater 計算當時週次
	- 快速事件紀錄：將事件記錄到`事記`檔案中，方便撰寫日誌時回想
4. 以上三個記錄方式皆透過 obsidian 命令觸發
